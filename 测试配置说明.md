## 修改说明

### 修改的属性

- 将 `promptText="xxx"` 改为 `text="xxx"`
  - `promptText`：占位符文本（输入框为空时显示的灰色提示）
  - `text`：实际文本（输入框中的默认值）

### 为什么这样修改

- 使用 `text` 属性可以让输入框直接显示默认值
- 方便测试时不需要手动输入 IP、用户名、密码
- 可以直接点击"测试"按钮进行连接测试

## 测试步骤

1. 启动应用程序
2. 界面会自动显示填好的 IP、用户名、密码
3. 直接点击"测试"按钮或"测试所有连接"
4. 如果虚拟机配置正确，测试会成功
5. 点击"下一步"继续后续流程

## 注意事项

- 确保你的虚拟机 IP 地址确实是 192.168.109.101/102/103
- 确保虚拟机上的用户名是 lyq，密码是 123456
- 确保虚拟机已启动并且 SSH 服务正在运行
- 确保宿主机和虚拟机网络互通

## 快速测试命令

在宿主机上测试网络连通性：

```bash
# 测试虚拟机1
ping 192.168.109.101

# 测试SSH连接
ssh lyq@192.168.109.101
```
